## ğŸ”‘ Key Concepts

### 1. The Tools
*   **GitHub Copilot:**
    *   Plugin for VS Code.
    *   Predicts the next few lines of code based on context.
    *   Learns your style.
*   **Cursor:**
    *   A full Code Editor (fork of VS Code).
    *   "Natural Language to Code": Type "Make a login screen" and it generates the files and code.
    *   Can refactor and fix bugs.

### 2. Benefits vs. Risks
*   **Benefits:** Faster speed, less boilerplate, explains complex code to beginners.
*   **Risks:**
    *   **Hallucinations:** AI might write buggy or insecure code.
    *   **Privacy:** Be careful sending company secrets to the cloud.
    *   **Over-reliance:** You still need to understand the *basics* to fix what the AI breaks.
