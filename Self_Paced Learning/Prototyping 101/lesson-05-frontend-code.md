## üîë Key Concepts

### 1. The Three Layers
*   **HTML (Structure):** The skeleton (Headings `<h1>`, Paragraphs `<p>`, Images `<img>`).
*   **CSS (Presentation):** The style (Colors, Fonts, Layouts). Can be Inline, Internal, or External.
*   **JavaScript (Behavior):** The interactivity (Form validation, Animations, API calls).

### 2. Responsive Design
*   Ensures sites look good on Mobile, Tablet, and Desktop.
*   **Media Queries:** CSS feature that detects screen size.
*   **Bootstrap:** A framework with a grid system for easy responsive layouts.

### 3. Progressive Web Apps (PWA)
*   Websites that behave like native mobile apps (can work offline, push notifications, add to home screen).
